# KVS Final Project

Рабочий репозиторий финальной работы по КВС.

Основные файлы:

- `KVS_Hebrew.docx` — финальная работа на иврите.
- `KVS_Russian.docx` — полный русский перевод для проверки.

Структура ивритской работы:

1. `זהות יהודית נגד זהות יוונית` — 2 страницы.
2. `קבוצות יהודים בימי בית שני` — 4 страницы:
   - около 1,5 страницы исторического введения;
   - около 2,5 страницы пьесы.

Формат: Arial 11, межстрочный интервал 1,15, поля 2 см. В документах нет примечаний, комментариев и отдельной титульной страницы.
